# AiVChat
企业微信/个人微信聚合消息 AI客服回复 同质化数据检索 私域运营CRM  
无需云端账号登录，PC端个人微信/企业微信继续保持在个人电脑登录

## 功能简介

多账号/多渠道消息聚合，AI回复，AI无法处理的消息，通知客服人工处理  
客户管理  CRM，自定义客户信息表单  
消息群发  
AI知识库，自研同质化数据AI检索引擎，解决当前知识库无法推荐海量SKU商品问题
简单AI流程编排，调用本地工具，复杂编排调用Coze平台工作流
